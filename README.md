# saplearninghub-pdf-downloader

Download eBook as PDF from SAP Learning Hub.

## Getting Started

## Installation

```bash
npm install
```

## Usage

![SAP Learning Hub](https://github.com/AZn5ReD/saplearninghub-pdf-downloader/blob/master/images/SAP_Learning_Hub.png?raw=true)

```bash
npm run start
```
